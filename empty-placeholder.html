Placeholder for empty lists using only HTML and CSS

Example of using the :empty pseudo-class together with a pseudo-element in CSS to display a placeholder message when an element does not have child nodes. Using HTML5 data attribute in CSS to decouple placeholder text from stylesheet for easy localization.
